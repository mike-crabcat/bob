# AI News Brief Journal

Purpose: track topics already covered by the daily AI news brief so future briefs avoid repeats without relying on WhatsApp bulletins/memory.

Rules:
- Before posting a brief, read this file.
- Treat each covered topic as a topic cluster, not just an exact headline. Example: “OpenAI IPO filing” covers follow-up articles about the same filing unless there is genuinely new material.
- Only include items not already covered here.
- After posting, append a dated entry with the topic names, one-line summaries, dates, and sources used.
- If this file grows above roughly 100 KB or becomes annoying to scan, split it: move older entries into `docs/ai-news-brief-journal-archive-YYYY-MM.md` and leave a short archive note here. Because apparently even journals need waste management.
